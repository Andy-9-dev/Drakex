# Quick Start Guide

## 🚀 Get Started in 3 Steps

### Step 1: Install Dependencies
```bash
cd Drakex-main
npm install
```

### Step 2: Start Development Server
```bash
npm run dev
```

### Step 3: Open in Browser
Visit `http://localhost:5173` in your browser

## 📱 What You'll See

The website includes 6 fully functional pages:

1. **Home** (`/`) - Beautiful hero section, stats, features, and testimonials
2. **Academics** (`/academics`) - Complete program listings for all degree levels
3. **Campus Life** (`/campus-life`) - Sports, arts, clubs, and recreational activities
4. **About Us** (`/about`) - University history, mission, values, and leadership
5. **Contact** (`/contact`) - Contact form and department information
6. **Registration** (`/registration`) - Full application form

## ✨ Key Features

## 🎨 Customization Tips

### Change Colors
Edit `tailwind.config.js` - look for the `primary` color definitions

### Update Content
All page content is in `src/pages/` - edit the JSX files directly

### Add Images
Place new images in the `img/` folder and update the image paths in components

### Modify Navigation
Edit `src/components/Navbar.jsx` to add/remove menu items
